# java-rest-api
This is demo application created to fetch user details from the other api.
Operations are :
1. get all users -  /users/
2. get user by id - /users/id
3. update - /users/id
4. delete - /users/

   API details :

   http://localhost:8080/users/

   Swagger doc - http://localhost:8080/swagger-ui.html
